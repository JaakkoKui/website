This page will host embedded builds (WebGL / HTML5 / etc.). For now it's a placeholder.

Upload your game build here.

How to Add a Build

  1. Create a folder under games/your-game-name/
  2. Export / copy your build files there (e.g., index.html, assets, build data)
  3. Link to it from this page or embed via iframe.

Example iframe snippet once you have a build:

<iframe src="games/your-game-name/index.html" title="Your Game" loading="lazy" allowfullscreen class="embed"></iframe>
